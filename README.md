Mercado Central

Mercado Central é uma aplicação web desenvolvida em ReactJS que simula um supermercado digital.

O projeto consome dados da API DummyJSON Products para exibir produtos, detalhes de cada item, filtros, busca, ordenação e carrinho de compras.

Sobre

Este projeto foi desenvolvido como atividade acadêmica da disciplina de ReactJS.

A proposta era criar uma aplicação em React que realizasse requisições em uma API externa, contendo pelo menos uma página de listagem e uma página de detalhes de um item selecionado.

A API escolhida foi a DummyJSON, utilizando a rota de produtos.

Documentação da API:

https://dummyjson.com/docs/products

Como rodar o projeto
Clone o repositório:

git clone https://github.com/SEU-USUARIO/mercado-central-react.git

Acesse a pasta do projeto:

cd mercado-central-react

Instale as dependências:

npm install

Rode o projeto:

npm run dev

Acesse no navegador:

http://localhost:5173

Rotas da aplicação
/ — página inicial com listagem de produtos
/produto/:id — página de detalhes do produto
/carrinho — página do carrinho
/sobre — página sobre o projeto
API utilizada

Lista de produtos:

https://dummyjson.com/products

Detalhes de produto por ID:

https://dummyjson.com/products/1
